Wrong time in websites

To fix this

  • Edit /etc/conf.d/hwclock
    • set clock="local" and run rc-service hwclock restart
    • now to set the time run hwclock --hctosys
  • Once this is done your time issue should be fixed
  • But date command will not display the local time, to do so run date -u everytime you run the date command.
