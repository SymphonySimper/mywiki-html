How to install mysql

  • Run emerge --config dev-db/mysql
    • The above command configures and installs mysql.
  • Run emerge acct-user/mysql acct-group/mysql

How to run

  • First make sure mysql server is running rc-service mysql status the output should say * status: started
    • If not start the service by running rc-service mysql start
    • If you don't want to do this everytime you can make mysql start at the boot by running rc-update add mysql default
      • This will start mysql at boot.
  • Run mysql --user=username --password=password
    • eg: mysql --user=root --password=root
    • In --user=root root is the username and in --password=root root is the password for the user root.
    • Username and password would be assignend by you when installing mysql with --config option as mentioned in #How to install mysql

Error 2002

Error looks like this 

	ERROR 2002 (HY000): Can't connect to local MySQL server through socket '/var/run/mysqld/mysqld.sock' (2)
  • To resolve this error do the following
    1. touch /var/run/mysqld/mysqld.sock
    2. touch /var/run/mysqld/mysqld.pid
    3. chown -R mysql:mysql /var/run/mysqld/mysqld.sock
    4. chown -R mysql:mysql /var/run/mysqld/mysqld.pid
    5. chmod -R 644 /var/run/mysqld/mysqld.sock
  • Once done start the service
    • Run rc-service mysql start
  • If everything goes well this error should be no more.
